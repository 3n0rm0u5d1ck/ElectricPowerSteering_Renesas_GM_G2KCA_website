0: Compare register mode.
1: Capture register mode.		 Can be written at field or register level
8
CTL0 CRM1 ENCA0CRM1 0 Register mode
0: Compare register mode.
1: Capture register mode.		 Can be written at field or register level
9
CTL MCS ENCA0MCS 0 Don't care Can be written at field or register level
10
CTL CME ENCA0CME 0 Clear mask enable
0: Disable compare interrupt.
1: Enable compare interrupt.		 Can be written at field or register level

IOC1 UNIT8 ENCA0IOC1
Register level access Can be written at field or register level
12
IOC1 EIS ENCA0EIS 0 Invalid for both edge detection (UDS = 11) Can be written at field or register level
13
IOC1 ECS ENCA0ECS 1 Rising edge detection. (Since we do not use Idx pin) Can be written at field or register level
14
IOC1 ACL ENCA0ACL 0 A - Clear condition select. (Don't care)
0: Clear Low Level.
1: High Level		 Can be written at field or register level
15
IOC1 BCL ENCA0BCL 0 B - Clear condition select. (Don’t care)
0: Clear Low Level.
1: High Level		 Can be written at field or register level
16
IOC1 ZCL ENCA0ZCL 0 Z - Clear condition select. (Don't care)
0: Clear Low Level.
1: High Level		 Can be written at field or register level
17
IOC1 SCE ENCA0SCE 0 Clear Enable
0: Clears counter upon valid edge on Z.
1: Clears counter upon valid edge on A B and Z.		 Can be written at field or register level
18
TS UNIT8 ENCA0TS
Register level access Can be written at field or register level
19
TS TS ENCA0TS 1 Enable counter. Set to operation. Can be written at field or register level
